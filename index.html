Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • DRY coding is crucial! Compress where you can.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period indactes class selector. Classes are arbitrary attributes inside HTML elements.

Git

  • $ git status: displays the current branch and its status
  • $ git checkout [flag] <branch-name>: navigates to the specified branch
    • -b: indicates you're creating a new branch
  • $ git pull origin <branch-name>: updates the working branch from a specified branch
  • $ git push origin <branch-name>: pushs commits from the working branch onto the specified branch. It is not recommended to use this as GitHub has tools built into the site to track and avoid errors
